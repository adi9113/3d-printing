# 3d-printing

All my 3D printing projects can be found in this repository.

## General Information

For my projects I am currently using the Creality Ender-3 S1 Pro 3D printer and UltiMaker Cura as slicer software. Since Cura did not provide any materials from Creality, I made my own base profile for the [Ender-PLA](/Cura-Files) which is heavily based on Cura's generic profile. I also made a few minor adjustments to the Cura profile for standard quality ([ender-profile](/Cura-Files) and [ender-speed-profile](/Cura-Files)). For 3D modeling I use OpenSCAD ([OpenSCAD-Tutorial](/OpenSCAD-Tutorial) / [Website of OpenSCAD Tutorial](https://en.wikibooks.org/wiki/OpenSCAD_Tutorial)).
