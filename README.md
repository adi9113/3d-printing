# 3d-printing

All my 3D printing projects can be found in this repository.

## General Information

For my projects I am currently using the Creality Ender-3 S1 Pro 3D printer and UltiMaker Cura as slicer software. Since Cura did not provide any materials from Creality, I made my own base profile for the [Ender-PLA](/cura-files/ender-pla.xml.fdm_material) which is heavily based on Cura's generic profile. I use openSCAD to develop the 3D models
